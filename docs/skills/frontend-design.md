# frontend-design

有目的性和辨识度的视觉设计指导。

## 触发条件

```
> 这个页面的设计看起来太模板化了，帮我改进
> 推荐一个字体组合
> 帮我选择配色方案
```

## 设计方向

- 🎨 美学方向选择
- 🔤 字体排版决策
- 🌈 色彩系统选择
- 📐 间距和布局节奏

## 常见风格

| 风格 | 特征 |
|------|------|
| Glassmorphism | 玻璃质感、模糊背景 |
| Claymorphism | 黏土效果、柔和阴影 |
| Minimalism | 极简、大量留白 |
| Brutalism | 粗犷、高对比度 |
| Neumorphism | 新拟物、柔和立体 |
