# MagicPath 插件

MagicPath 插件提供 UI 组件搜索、安装和主题管理能力，帮助快速找到适合的 UI 组件并将其集成到项目中。

## 功能概述

- 🔍 UI 组件搜索和发现
- 📦 组件安装和管理
- 🎨 主题和设计管理
- 🖌️ 画布选择操作
- 🔄 仓库到画布 UI 工作流

## 包含的技能

### 技能列表

| 技能名称 | 描述 |
|---------|------|
| `magicpath` | 使用 MagicPath 搜索、检查、安装或编写组件 |

## MagicPath 技能 (`magicpath`)

当用户提到 MagicPath、设计、UI 组件、主题、画布选择或仓库到画布 UI 工作时使用。

### 触发时机

- 搜索 UI 组件
- 安装组件到项目
- 管理主题和设计方案
- 处理画布选择
- 从仓库 UI 创建画布

## 工作流程

### 组件搜索与安装

1. **搜索**：根据需求在 MagicPath 注册表中搜索合适的组件
2. **检查**：预览组件的样式、API 和兼容性
3. **安装**：将组件添加到项目依赖
4. **定制**：根据项目需求调整组件样式

### 主题管理

1. **选择**：浏览和选择预设主题
2. **配置**：调整颜色、字体、间距等设计令牌
3. **应用**：将主题应用到整个项目或特定组件

---

## 最佳实践

- 安装前检查组件的许可证兼容性
- 优先选择与项目已有设计系统一致的组件
- 安装后根据需要覆盖样式以匹配项目品牌
- 定期更新组件以获取安全修复和新功能
