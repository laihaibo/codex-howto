# hyperframes-registry

HyperFrames 组件块注册表的安装和管理。

## 触发条件

```
> 安装一个 hyperframes 组件
> 添加背景动画到我的合成
> 查看可用的组件块
```

## 主要命令

- `hyperframes add <package>` - 安装和接线组件块
- `hyperframes search <keyword>` - 搜索可用组件
- `hyperframes list` - 显示已安装组件

## 安装流程

1. 搜索和发现可用组件块
2. 安装到项目
3. 自动接线到 index.html
4. 配置和自定义

## 组件类型

| 类型 | 说明 |
|------|------|
| block | 子组合（完整场景） |
| component | 可复用组件片段 |
