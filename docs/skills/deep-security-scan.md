# deep-security-scan

深度安全扫描技能，执行穷尽式的多轮安全分析。

## 触发条件

```
> 对这个仓库运行深度安全扫描
> 分析这个项目的所有安全问题
> 彻底检查我的代码是否存在漏洞
```

## 工作机制

1. **多轮独立发现**：多个 worker 从不同角度分别扫描
2. **语义合并**：智能去重和合并相似发现
3. **威胁建模**：生成统一验证的威胁模型
4. **一轮验证+攻击路径分析**：集中验证所有发现
5. **生成综合报告**

## 适用场景

- 关键项目的发布前安全审查
- 合规要求的全面安全审计
- 外部安全评估的预检

## 与标准扫描的区别

| 特性 | 标准扫描 | 深度扫描 |
|------|---------|---------|
| 扫描轮次 | 1 轮 | 多轮 |
| Worker 数量 | 1 | 多个 |
| 威胁模型 | 基础 | 综合 |
| 发现覆盖 | 标准 | 穷尽 |
