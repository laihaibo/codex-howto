# brainstorming

创意工作前的需求探索和设计思考技能。

## 触发条件

```
> 用 brainstorming 探索这个支付模块的设计方案
> 这个功能应该怎么做？先 brainstorming 一下
```

## 工作流程

1. **探索意图**：理解用户真正想要什么
2. **收集需求**：功能、非功能、约束条件
3. **发现设计选择**：多种可行方案对比
4. **确认方向**：选择最合适的实现路径

## 适用场景

- 🆕 创建新功能前
- 🏗️ 构建新组件前
- 🔄 修改行为前
- 🎨 设计 UI/UX 前

## 输出

澄清后的需求文档和方案设计，为后续 $writing-plans 做准备。
