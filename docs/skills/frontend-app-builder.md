# frontend-app-builder

前端应用构建技能，用于从零创建完整的前端应用、仪表盘、游戏、创意网站等视觉驱动的界面。

## 触发条件

- 创建新的前端应用或页面
- 需要构建仪表盘、落地页、管理后台
- 重新设计或现代化现有 UI
- 实现设计师提供的设计稿

## 工作流程

1. **概念设计**：使用图像生成创建视觉概念
2. **详细设计**：为每个部分生成特定参考图
3. **忠实实现**：精确还原设计稿的代码实现
4. **浏览器测试**：验证渲染效果和交互

## 示例提示词

```
> 创建一个 SaaS 产品的落地页，包含 Hero 区、功能特性、定价表和客户评价
> 用这个设计稿重新实现我们的设置页面
> 创建一个暗色主题的管理仪表盘
```

## 最佳实践

- 提供设计稿或参考图片时效果最佳
- 明确目标设备和响应式需求
- 指定偏好的技术栈（React、Vue 等）
- 分步构建，逐个区块实现
