# dispatching-parallel-agents

并行处理多个独立任务的能力。

## 触发条件

```
> 并行处理这些独立任务
> $parallel-agents 实现这三个独立模块
```

## 适用场景

- 2+ 个独立任务，无共享状态
- 任务间没有顺序依赖
- 每个任务可独立分配

## 调度策略

| 场景 | 处理方式 |
|------|---------|
| 完全独立 | 并行启动多个 worker |
| 有依赖关系 | 先执行依赖任务，再并行独立任务 |
| 需协商结果 | 先收集结果，再统一协调 |

## 注意事项

- 确保 worker 之间不会产生文件冲突
- 每个 worker 有明确的任务边界
- 合并结果时处理可能的冲突
