# hyperframes

HyperFrames 技能套件，用于创建 HTML 视频合成和动画。

## 触发条件

```
> 创建一段产品宣传视频
> 给这段文字添加动画效果
> 用 GSAP 制作 logo 揭示动画
```

## 技能系列

| 技能 | 用途 |
|------|------|
| hyperframes | 核心合成和动画制作 |
| hyperframes-cli | CLI 工具操作 |
| hyperframes-registry | 组件块安装 |
| website-to-hyperframes | 网站转视频 |
| gsap | 动画参考 |

## 工作流程

1. 创建 HTML 合成（场景、元素、时间线）
2. 添加动画和过渡效果
3. 配置音频/字幕
4. 预览效果
5. 渲染为视频文件
