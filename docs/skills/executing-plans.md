# executing-plans

分步执行实施计划，每个步骤设有审查检查点。

## 触发条件

```
> 执行这个实施计划
> $executing-plans 按计划实现功能
```

## 执行流程

1. **读取计划**：理解完整的实施步骤
2. **逐步执行**：按计划顺序实现每个步骤
3. **检查点审查**：每个步骤完成后暂停确认
4. **问题处理**：发现问题时调整计划
5. **完成验证**：所有步骤完成后总结

## 检查点项目

- ✅ 代码是否按预期工作
- ✅ 测试是否全部通过
- ✅ 是否符合计划中的技术规范
- ✅ 是否需要调整后续步骤
