# Obsidian 技能

Obsidian 笔记管理的综合技能套件。

## 包含技能

### obsidian-cli

通过 CLI 与 Obsidian 保险库交互：

```
> 读取我的笔记：projects/my-project
> 搜索包含 "React hooks" 的笔记
> 创建一个任务清单笔记
> 运行插件 JavaScript
```

### obsidian-markdown

创建和编辑 Obsidian Flavored Markdown：

- Wikilinks `[[Note Name]]`
- 嵌入内容 `![[Note]]`
- 呼出块 `> [!note]`
- 属性 frontmatter
- Tags `#tag`

### obsidian-bases

创建 Obsidian Bases（数据库视图）：

```
> 创建一个笔记的卡片视图
> 添加过滤器和公式
> 制作笔记的表格视图
```
