# code-reviewer

代码审查技能，支持本地更改（暂存或工作树）和远程 Pull Request。

## 触发条件

```
> 审查当前暂存区的代码变更
> 审查 PR #123 的代码
> 审查这个 URL 的 PR
```

## 审查维度

| 维度 | 检查项 |
|------|--------|
| 正确性 | 逻辑是否正确，边界条件处理 |
| 可维护性 | 代码结构、命名、复杂度 |
| 安全性 | 漏洞风险、数据验证 |
| 风格 | 符合项目约定和最佳实践 |

## 审查输出

- 行级问题定位
- 严重性评级（高/中/低）
- 修复建议
- 优点认可
