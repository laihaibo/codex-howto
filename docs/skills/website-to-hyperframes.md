# website-to-hyperframes

从现有网站创建宣传视频或产品演示。

## 触发条件

```
> 用这个网站创建一段宣传视频
> 捕获 google.com 制作产品介绍
> 把这个链接变成一段演示视频
```

## 使用方式

只需提供一个 URL，Codex 会：

1. 捕获网站内容
2. 创建 HyperFrames 合成
3. 添加过渡动画
4. 生成视频

## 示例

```
> 用这个 landing page 制作一段 15 秒的产品宣传视频
> https://mysaas.com → 创建介绍视频
```
