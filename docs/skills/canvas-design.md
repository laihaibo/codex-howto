# canvas-design

使用设计理念创建 PNG 和 PDF 格式的美丽视觉艺术作品。

## 触发条件

```
> 创建一个海报设计
> 制作一张活动邀请函
> 生成品牌标识的 PDF
> 制作社交媒体图片
```

## 适用场景
- 海报设计
- 名片、传单
- 社交媒体图
- 品牌视觉
- 原创艺术作品

## 输出格式

- `.png` - 位图格式
- `.pdf` - 文档格式

::: warning 注意
仅创建原创设计，不复制现有作品。
:::
