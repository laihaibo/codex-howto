# remotion

Remotion 视频创建最佳实践——在 React 中创建视频。

## 触发条件

```
> 用 Remotion 创建一段倒计时视频
> 制作一个动态数据展示视频
> 用 React 渲染视频帧
```

## 核心概念

- 以编程方式生成视频
- React 组件定义视频内容
- 帧精确控制
- 支持动画、音频、字幕

## 与 HyperFrames 的区别

| 特性 | Remotion | HyperFrames |
|------|---------|-------------|
| 输出格式 | MP4/WebM | MP4/WebM |
| 编程方式 | React 组件 | HTML/CSS |
| 音频 | FFmpeg | Web Audio |
| 适用 | 数据视频、动画 | 宣传视频、网页动画 |
