# using-superpowers

Superpowers 使用入门指南——学习如何触发和使用技能。

## 核心概念

Superpowers 是一组可扩展的工作流技能，通过 `$SkillName` 触发。

## 触发方式

```
> 开始 brainstorming 这个新功能的设计
> $writing-plans --scope "实现 OAuth 登录"
> 用 TDD 实现用户注册模块
```

## 技能发现

在对话中询问可用技能列表，或参考技能目录：

```
> 列出所有可用的 superpowers 技能
> 有哪些工作流技能可以使用？
```

## 使用原则

1. **先思考再动手**：creative 工作前先 brainstorm
2. **复杂任务先规划**：多步骤任务使用 writing-plans
3. **追求质量**：使用 TDD 和系统化调试
4. **团队协作**：有效请求和接收代码审查
5. **保持验证**：完成工作前验证结果
