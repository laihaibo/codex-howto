# data-visualization

Web 数据可视化路由技能，帮助选择合适的图表类型并实现可视化。

## 触发条件

- 需要选择合适图表类型
- 设计可视化仪表盘
- 创建地图或地理空间视图
- 制作甘特图时间线
- 绘制 UML/软件架构图
- 生成可滚动的可视化故事
- 导出报告

## 图表类型决策表

| 数据类型 | 推荐图表 |
|---------|---------|
| 对比/排名 | 柱状图、条形图 |
| 时间趋势 | 折线图、面积图 |
| 占比关系 | 饼图、环形图、堆叠图 |
| 分布 | 直方图、箱线图 |
| 相关性 | 散点图、气泡图 |
| 流程 | 桑基图、流程图 |
| 地理 | 地图、热力图 |

## 示例提示词

```
> 创建一个销售数据的仪表盘，包含趋势图和分类占比
> 用地图展示全球用户分布
> 这个数据集应该用什么图表展示最好？
> 把这份数据导出为 PDF 报告
```

## 设计原则

- 选择最匹配数据特征的图表类型
- 确保可访问性（颜色对比度、alt 文本）
- 响应式适配不同屏幕
- 避免过度装饰（data-ink ratio）

## 测试策略

- 浏览器中验证渲染效果
- 检查响应式布局
- 确认无障碍属性
