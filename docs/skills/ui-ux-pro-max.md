# ui-ux-pro-max

全面的 Web 和移动端 UI/UX 设计智能系统。

## 设计能力

- 🎨 **50+ 种风格**：Glassmorphism、Claymorphism、Minimalism 等
- 🌈 **161 个色板**：精心策划的色彩系统
- 🔤 **57 个字体组合**：优化的字体搭配
- 📦 **161 种产品类型**：场景化设计参考
- 📏 **99 条 UX 指南**：经过验证的用户体验原则
- 📊 **25 种图表类型**：数据可视化最佳实践

## 支持的 10 大技术栈

React | Next.js | Vue | Svelte | SwiftUI | React Native | Flutter | Tailwind | shadcn/ui | HTML/CSS

## 适用场景

- 网站和应用设计
- Landing page、Dashboard、管理面板
- 电商、SaaS、作品集、博客
- Button、Modal、导航、侧边栏、卡片等组件
