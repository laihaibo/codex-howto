# imagegen - 图像生成

图像生成技能，允许 Codex 通过 AI 创建或编辑光栅（raster）图像。生成的图像以位图格式输出，适用于照片、插图、纹理、精灵图和透明背景剪贴画等场景。

## 何时使用

以下场景适合触发 `imagegen` 技能：

- 需要 AI 创建的原创位图视觉内容
- 从参考图像派生视觉变体
- 生成位图资产（而非代码或矢量图）

### 适用场景

| 场景 | 说明 |
|------|------|
| **照片** | 根据描述生成逼真的照片风格图像 |
| **插图** | 绘制数字插画和艺术作品 |
| **纹理/贴图** | 生成材质纹理和背景图案 |
| **精灵图/Sprites** | 创建游戏或 UI 用的精灵图元素 |
| **透明剪贴画** | 生成带透明背景的切割图 |
| **Mockup 模型** | 创建产品展示模型和样机 |
| **海报/艺术** | 创作宣传海报和视觉艺术 |

### 不适用场景

以下情况请**不要**使用 `imagegen`：

- **SVG/矢量图** — 使用代码或矢量编辑工具直接编写
- **代码原生元素** — 通过 HTML/CSS/Canvas 实现的视觉效果
- **已有图片系统的扩展** — 在已有图标或 Logo 体系上进行延展
- **文档图表** — 需要可编辑、可缩放的矢量格式

## 使用示例

### 基础生成

```
请帮我生成一张日落海滩的照片
创建一个未来风格的城市夜景插图
我需要一张木纹材质纹理，用于 3D 渲染
```

### 编辑变换

```
将这张照片调整为赛博朋克风格
把背景改成透明
生成这个设计的深色模式变体
```

### 变化与迭代

```
基于这个概念，给我 3 种不同的配色方案
再生成几个角度变化的版本
把这个风格应用到其他产品上
```

### 特定资产类型

```
生成一个游戏角色精灵图，64x64 像素，带透明背景
创建一个 App 图标样机，展示在 iPhone 上
制作一张社交媒体宣传海报，1080x1080
```

## 输出格式

`imagegen` 生成的图像为位图格式（PNG/JPEG/WebP），具体格式取决于使用场景：

- 需要透明背景 → PNG
- 照片类内容 → JPEG
- 兼顾质量与大小 → WebP

## 最佳实践

1. **描述具体** — 越详细的描述越能得到精确的图像
2. **指定风格** — 说明艺术风格（写实、扁平、像素艺术等）
3. **说明用途** — 告知图像将用于何处（网页、印刷、社交媒体等）
4. **提供参考** — 如有参考图片，上传后描述想要的变换
