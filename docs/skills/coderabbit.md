# coderabbit

使用 CodeRabbit AI 进行自动化代码审查。

## 触发条件

```
> 用 CodeRabbit 审查这个 PR
> 请求 CodeRabbit 代码审查
> AI 代码审查这个提交
```

## 功能

- 使用 AI 自动审查代码变更
- PR 级别的反馈（评论、建议）
- 代码质量检查
- 安全问题检测
- 修复-审查循环

## 适用场景

- 快速得到 PR 的初步反馈
- 发现潜在 bug 和代码异味
- 作为人工审查的补充
