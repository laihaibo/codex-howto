# magicpath

MagicPath UI 组件发现、安装和创作技能。

## 触发条件

```
> 搜索一个日期选择器组件
> 安装 MagicPath 的主题
> 创建自定义 UI 组件
> repo-to-canvas 工作流
```

## 功能

- 🔍 搜索 UI 组件
- 📦 安装组件到项目
- 🎨 主题和画布选择
- ✏️ 自定义组件创作

## 工具

通过 `magicpath-ai` CLI 运行：搜索、检查、安装或创作组件。

## 适用场景

- 快速找到合适的 UI 组件
- 从设计稿到代码画布
- 组件主题定制
- 设计系统同步
