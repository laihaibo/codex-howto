# writing-plans

将需求转化为多步骤实现计划的技能。

## 触发条件

```
> 为这个功能写实现计划
> $writing-plans "实现实时通知系统"
> 创建一个详细的开发计划
```

## 计划结构

```markdown
# 实现计划：功能名称

## 背景
简要描述问题和解决方案

## 实施步骤
1. [ ] 第一步：数据模型
2. [ ] 第二步：API 端点
3. [ ] 第三步：前端组件
4. [ ] 第四步：集成测试

## 技术决策
- 框架选择：...
- 数据存储：...

## 风险和注意事项
- 潜在性能问题：...
```

## 与 executing-plans 配合

写完计划后使用 `$executing-plans` 分步执行，每个步骤设检查点。
