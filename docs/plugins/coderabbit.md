# CodeRabbit 插件

CodeRabbit 插件提供基于 AI 的自动化代码审查能力，帮助在 Pull Request 中识别问题、提供改进建议并提升代码质量。

## 功能概述

- 🤖 AI 驱动的自动化代码审查
- 📝 PR 反馈和建议生成
- 🔄 修复-审查循环
- 🎯 代码质量和安全问题的识别

## 包含的技能

### 技能列表

| 技能名称 | 描述 |
|---------|------|

## CodeRabbit 代码审查 (`coderabbit-review`)

使用 CodeRabbit AI 审查代码，支持本地变更（暂存或工作树）和远程 Pull Request。

### 触发时机

- 用户要求代码审查
- 需要 PR 反馈
- 检查代码质量问题
- 安全审查
- 请求修复-审查循环

### 审查范围

CodeRabbit 可以审查：

- **正确性**：逻辑错误、边界条件、异常处理
- **可维护性**：代码复杂度、重复代码、命名规范
- **安全性**：注入漏洞、敏感信息泄露、不安全的依赖
- **性能**：低效算法、内存泄漏、不必要的重渲染

### 示例提示词

```
审查这个 PR 的代码变更
```

```
帮我做一次代码审查，重点关注安全性和性能
```

```
对 src/auth/ 目录下的变更进行代码审查
```

```
重新审查我修复后的代码
```

### 工作流程

1. 识别需要审查的代码范围（本地文件或远程 PR）
2. CodeRabbit AI 分析代码变更
3. 生成带有具体行号引用的审查评论
4. 用户可以逐条处理反馈或选择性地忽略

### 修复-审查循环

支持迭代式改进：

1. 收到 CodeRabbit 的审查反馈
2. 实施修复
3. 请求重新审查
4. 直到所有关键问题解决
